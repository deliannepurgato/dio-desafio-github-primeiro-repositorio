Após Atualizar ou criar pastas na máquina, é necessario enviar para o repositório remoto através do Gitbash com os seguintes processos:


Abrir o gitBash onde se encontra o diretório do .Git e realizar os seguintes comandos:

git add .
git commit -m "explicação do motivo da atualização ou criação"
git push origin main

Após esses processos, todos arquivos criados ou alterados serão atualizados no GitHub.
